Приложение имеет следующие функции
1) добавление заметки в базу данных
2) удаление заметки из базы данных
3) редактирование заметки в базе данных
4) есть возможность фильтрации списка заметок по дате
5) Реализован поиск полнотекстовый поиск заметок по названию и тексту заметок
6) Пользователь имеет возможность создавать произвольное кол-во заметок
7) База данных хранит результаты работы с заметками
8) Приложение реализовано в виде REST API, запросы лучше делать через PostMan
9) Использовались минимальные возможности javadoc
10) Поиск по #хэштегам не реализован

Для подключения к базе данных нужно прописать переменные среды
datasource.username=${DATA_BASE_USERNAME}
datasource.password=${DATA_BASE_PASSWORD}
datasource.url=${DATA_BASE_URL}

HTTP запросы для работы с приложением
1. Создание заметки
POST http://"hostname"/create/note
передать Body JSON
{
    "name":"название заметки",
    "note":"текст заметки",
}
2. Удаление заметки
DELETE http://"hostname"/delete/note/"название заметки"
3. Изменить заметку
PUT http://"hostname"/update/note/"название заметки"
в BODY написать текст заметки
4. Отфильтровать заметки по дате создания указав минимальное время и максимальное, если время не указано выдаст все заметки
GET http://"hostname"/find/notes/by-time?min="год"-"месяц"-"день"T"часы:минуты"&max="год"-"месяц"-"день"T"часы:минуты"
5. Найти заметки по ключевым словам в названии и в текстке заметки. При написание нескольких слов в параметре, вместо пробела использовать нижнее подчеркивание
GET http://"hostname"/find/notes/by-text?text="текст"
